# AK Studios

A professional storefront demo built to showcase a clean separation between a high-performance **React/Vite** frontend and a robust **ASP.NET** microservices backend.

### The Stack
* **Frontend:** React + Vite (Custom component architecture, dynamic image handling).
* **Backend:** ASP.NET API (Service-oriented architecture for inventory and checkout).

### Highlights
* **Performance-First:** Custom `fit`/`full` image loading logic to keep the UI snappy.
* **Component-Driven:** Modular React components for a seamless gallery and shopping experience.
* **API-Integrated:** Real-world data flow between client and server.

*Built as a portfolio project to demonstrate full-stack capabilities.*
